# Tomato Camera Stereo Pipeline

This package runs the stereo camera stack for the tomato harvesting robot.

It publishes raw camera images, calibrated camera info, rectified images, and stereo disparity.

---

## Camera Topics

### Raw Images

```text
/stereo/left/image_raw
/stereo/right/image_raw
```

Type:

```text
sensor_msgs/msg/Image
```

---

### Camera Info

```text
/stereo/left/camera_info
/stereo/right/camera_info
```

Type:

```text
sensor_msgs/msg/CameraInfo
```

These are loaded from calibration YAML files.

---

### Rectified Images

```text
/stereo/left/image_rect
/stereo/right/image_rect
```

Type:

```text
sensor_msgs/msg/Image
```

These are generated by `image_proc/rectify_node`.

---

### Disparity

```text
/stereo/disparity
```

Type:

```text
stereo_msgs/msg/DisparityImage
```

This is generated by `stereo_image_proc/disparity_node`.

---

## Start the Stereo Cameras

Launch the stereo stack:

```bash
ros2 launch tomato_bringup stereo.launch.py
```

Verify topics:

```bash
ros2 topic list | grep stereo
```

Check image streams:

```bash
ros2 topic hz /stereo/left/image_raw
ros2 topic hz /stereo/right/image_raw
```

Check calibration info:

```bash
ros2 topic echo /stereo/left/camera_info --once
ros2 topic echo /stereo/right/camera_info --once
```

---

## Run Rectification

Left camera:

```bash
ros2 run image_proc rectify_node \
  --ros-args \
  -r image:=/stereo/left/image_raw \
  -r camera_info:=/stereo/left/camera_info \
  -r image_rect:=/stereo/left/image_rect
```

Right camera:

```bash
ros2 run image_proc rectify_node \
  --ros-args \
  -r image:=/stereo/right/image_raw \
  -r camera_info:=/stereo/right/camera_info \
  -r image_rect:=/stereo/right/image_rect
```

Verify:

```bash
ros2 topic hz /stereo/left/image_rect
ros2 topic hz /stereo/right/image_rect
```

---

## Run Disparity

```bash
ros2 run stereo_image_proc disparity_node \
  --ros-args \
  -p approximate_sync:=true \
  -p queue_size:=10 \
  -p min_disparity:=0 \
  -p max_disparity:=64 \
  -p block_size:=15 \
  -p uniqueness_ratio:=10.0 \
  -p speckle_size:=100 \
  -p speckle_range:=4 \
  -p disp12_max_diff:=1 \
  -r left/image_rect:=/stereo/left/image_rect \
  -r left/camera_info:=/stereo/left/camera_info \
  -r right/image_rect:=/stereo/right/image_rect \
  -r right/camera_info:=/stereo/right/camera_info \
  -r disparity:=/stereo/disparity
```

Verify:

```bash
ros2 topic info /stereo/disparity
ros2 topic hz /stereo/disparity
```

## Visualize Raw or Rectified Images

Use `rqt`:

```bash
rqt
```

Open:

```text
Plugins → Visualization → Image View
```

Useful topics:

```text
/stereo/left/image_raw
/stereo/right/image_raw
/stereo/left/image_rect
/stereo/right/image_rect
```

To show both images in one `rqt` window, open two Image View panels.

---

## Visualize Stereo Disparity

Use `stereo_view`:

```bash
ros2 run image_view stereo_view \
  --ros-args \
  -p approximate_sync:=true \
  -r /stereo/left/image:=/stereo/left/image_rect \
  -r /stereo/right/image:=/stereo/right/image_rect \
  -r /stereo/disparity:=/stereo/disparity
```

This displays:

```text
left rectified image
right rectified image
disparity image
```
